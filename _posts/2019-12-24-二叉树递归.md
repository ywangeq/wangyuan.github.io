> 中等等级的题型

- 1.求根到叶子节点的数字之和
- 2. 二叉树的锯齿遍历




**题目1**
> 给定一个二叉树,每条从根到叶子结点的路径代表一个数字
> [1,2,3] output:25

```
def sumN(root):
    self.res=0
    def dfs(root,temp):
        if not root: return
        if not root.left and not root.right:
            self.res = int(temp+str(root.val))
            return
        temp +=str(root.val)
        dfs(root.left,temp)
        dfs(root.right,temp)
    dfs(root,'')
    return self.res
```

**2. 二叉树的锯齿遍历**
> 无脑递归+ 判断插入点+ 深搜

```
def zigzag(root):
    res=[]
    def dfs(root,depth):
        if not root: return
        if len(res)==depth: res.append([])
        if depth%2==0:
            res[depth].append(root.val)
        else:
            res[depth].insert(0,root.val)
        dfs(root.left,depth+1)
        dfs(root.right,depth+1)
    dfs(root,0)
    return res
```
