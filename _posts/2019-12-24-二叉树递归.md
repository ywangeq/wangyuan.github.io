> 中等等级的题型

- 1.求根到叶子节点的数字之和





**题目1**
> 给定一个二叉树,每条从根到叶子结点的路径代表一个数字
> [1,2,3] output:25

```
def sumN(root):
    self.res=0
    def dfs(root,temp):
        if not root: return
        if not root.left and not root.right:
            self.res = int(temp+str(root.val))
            return
        temp +=str(root.val)
        dfs(root.left,temp)
        dfs(root.right,temp)
    dfs(root,'')
    return self.res
```
