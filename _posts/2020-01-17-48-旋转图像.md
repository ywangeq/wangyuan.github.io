如题 旋转90的图像 顺时针
- 我们可以总结出规律 先转置矩阵，在反转每一行

```
def rotate(matrix):
    for i in range(len(matrix)):
        for j in range(i,len(matrix[0])):
            matrix[i][j],matrix[j][i] = matrix[j][i],matrix[i][j]
    for row in matrix:
        row.reverse()
    return matrix
```
