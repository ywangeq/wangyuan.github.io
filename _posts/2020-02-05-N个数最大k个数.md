N个数找最大k个数，
- 基于快排很好做,不需要排序整体，O(N)

```
def qicksel(A,k):
    if len(A)<k :
        return A
    pivot = A[-1]
    right = [pivot] + [ x for x in A[:-1] if x >=pivot]
    rlen =len(right)
    if rlen==k:
        return right
    if rlen>k:
        return qicksel(right,k)
    else:
        left = [x for x in A[:-1] if x < pivot]
        retun qicksel(left,k-rlen)+right
```

- 其次基于最大堆 O(NlogK)
- 我们创建一个大小为K的容器来存储k个数，遍历整个数组，将每个数字和容器中的最大数进行比较，如果这个数大于容器中的最大值，则继续遍历，不然替换掉这个最大值
